# 🛠️ 文本处理工具箱 | Text Processing Toolkit

一个集成了多种实用功能的在线文本处理工具集合。纯前端实现，无需后端，快速、安全、即开即用。

**在线直达链接：🔗 [https://wenzi.netlify.app/](https://wenzi.netlify.app/)**

---

## ✨ 核心功能

### 1. 📊 智能字数统计
- **实时统计**：输入时即时更新所有统计数据。
- **多维分析**：
  - 字符数（含空格/不含空格）
  - 中文字符数 & 英文单词数
  - 总行数与段落数
- **设计亮点**：采用防抖(`debounce`)技术优化性能，避免高频计算卡顿。

### 2. 🔠 大小写转换器
- **一键转换**：支持全部大写、全部小写、首字母大写。
- **便捷输出**：提供结果输出框与“一键复制”按钮，快速获取结果。

### 3. 🧹 文本去重工具
- **按行去重**：自动识别并移除重复的行内容。
- **数据对比**：清晰展示原始行数、去重后行数及去除的重复行数。

### 4. ⚖️ 文本对比工具
- **并排对比**：直观的双栏布局，方便查看两段文本差异。
- **差异高亮**：实时高亮显示内容不同的部分，快速定位变更。
- **视觉优化**：差异区域有明确的背景色标识。

### 5. 🧩 JSON格式化与压缩
- **美化格式化**：将压缩的JSON字符串转换为结构清晰、带缩进的格式。
- **反向压缩**：将格式化的JSON压缩为单行字符串，节省空间。
- **语法验证**：自动检测JSON格式是否有效，并给出友好错误提示。

---

## 🖥️ 技术栈与实现

- **核心三件套**：原生 `HTML5`、`CSS3`、`JavaScript (ES6+)`
- **布局方案**：采用 `CSS Grid` 与 `Flexbox` 构建的完全响应式布局。
- **交互逻辑**：纯原生JavaScript实现，无任何外部框架依赖，轻量高效。
- **代码架构**：
  - 使用 `CSS Custom Properties (变量)` 统一管理主题色。
  - JavaScript函数模块化，每个工具功能独立封装，便于维护。
- **部署**：托管于 **Netlify**，实现自动化部署与全球CDN加速。
